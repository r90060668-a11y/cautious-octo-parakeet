# Kiro Arcade

Android-приложение на Jetpack Compose с нижней панелью в стиле «жидкого стекла»
(эффект размытия/переливания через библиотеку [Haze](https://github.com/chrisbanes/haze))
и «ползунком», который нужно перетаскивать, чтобы переключить вкладку — как в Telegram на iPhone.

## Что внутри

- 4 вкладки: **Посты**, **Чаты**, **Профиль**, **Хосты**
- Каждая вкладка пустая и показывает текст:
  «На данный момент происходит тест, приложение не работает»
- Переключение вкладок — свайпом по экрану **или** перетаскиванием стеклянного
  «ползунка» внутри нижней панели, либо тапом по иконке
- Нижняя панель — плавающая капсула с настоящим блюром фона (Android 12+/API 31+;
  на более старых версиях библиотека автоматически откатывается на полупрозрачную заливку)

## Как открыть и запустить

1. Установи **Android Studio** (Ladybug/Koala или новее).
2. Открой папку `KiroArcade` через `File → Open`.
3. Studio сама предложит скачать Gradle Wrapper — согласись (нужен интернет один раз).
   Если Wrapper не подтянулся автоматически, выполни в терминале внутри папки проекта:
   ```
   gradle wrapper --gradle-version 8.9
   ```
   (требуется установленный Gradle — просто как разовая операция).
4. Дождись синхронизации Gradle (подтянутся зависимости Compose и Haze).
5. Нажми ▶️ Run на эмуляторе или реальном устройстве с **Android 8.0 (API 26) и выше**.
   Для полноценного эффекта размытия стекла нужен **Android 12 (API 31)+**.

## Структура проекта

```
app/src/main/java/com/kiro/arcade/
├── MainActivity.kt        # вся логика UI: пейджер вкладок, стеклянная панель, ползунок
└── ui/theme/               # цвета и тема Compose
```

## Что можно докрутить дальше

- Заменить пустышки вкладок на реальный контент
- Добавить иконку приложения с собственным лого (сейчас — заглушка)
- Настроить интенсивность блюра через `HazeMaterials.thin() → .regular()/.thick()`
  в `GlassBottomBar` внутри `MainActivity.kt`
