# Kiro Arcade

Android-приложение на Jetpack Compose с нижней панелью в стиле «жидкого стекла»
(эффект размытия/переливания через библиотеку [Haze](https://github.com/chrisbanes/haze))
и «ползунком», который нужно перетаскивать, чтобы переключить вкладку — как в Telegram на iPhone.

## Что внутри

- 4 вкладки: **Посты**, **Чаты**, **Профиль**, **Хосты**
- Каждая вкладка пустая и показывает текст:
  «На данный момент происходит тест, приложение не работает»
- Переключение вкладок — свайпом по экрану **или** перетаскиванием стеклянного
  «ползунка» внутри нижней панели, либо тапом по иконке
- Нижняя панель — плавающая капсула с настоящим блюром фона (Android 12+/API 31+;
  на более старых версиях библиотека автоматически откатывается на полупрозрачную заливку)
